# Tachidesk-qtui
a c++/qt client app for use with Tachidesk server

## what works:
- selecting manga and viewing and selecting chapters
- installing/uninstalling/updaing extensions
- viewing latest library from sources
- adding/removing manga
- basic settings

## what doesn't work:
- no backup/restore
